## 🏗️ Estrutura do Portfólio

O aprendizado foi dividido em três grandes trilhas de especialização:

### 🔹 Trilha 1: Fundamentos (Exercícios 01 a 30)
*   **Foco:** Lógica de programação e JavaScript puro (Vanilla JS).
*   **Conceitos:** Manipulação do DOM, funções, condicionais, loops e fundamentos da Web.

### 🔹 Trilha 2: React & Componentização (Exercícios 31 a 70)
*   **Foco:** Construção de interfaces modernas e reativas.
*   **Conceitos:** JSX, Props, `useState`, Elevação de Estado (Lifting State Up), Componentes Controlados e Acessibilidade.
*   **Projetos de destaque:** Admin de Loja Online e App de Afazeres.

### 🔹 Trilha 3: Avançado & Integração com APIs (Exercícios 71 a 100)
*   **Foco:** Aplicações escaláveis e comunicação com o servidor.
*   **Conceitos:** `useEffect`, Ciclo de Vida, Persistência com `localStorage`, Consumo de APIs REST (Fetch/Async-Await) e Custom Hooks.
*   **Projetos de destaque:** Localizador de Lojas, Integração com Mapas e Painel Administrativo.
